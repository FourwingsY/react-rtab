react-rtab

React tab-panel component.

Description

react-rtab use Models to rendering tab component.

Models should be an array of Model {tab: TabModel, panel: PanelModel}

TabModel could be an object with renderer, or plain text. PanelModel could be an object with renderer, Component, or Component instance.
