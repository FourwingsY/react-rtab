React-rtab

React tab-panel component.

Description

react-rtab use Models to repeatedly rendering tab component.

Models should be an array of Model {tab: TabModel, panel: PanelModel}
TabModel could be an object with renderer, or plain text.
PanelModel could be an object with renderer, ReactElement, or getter function.

You can turn on draggable or closable options. Try to drag and close tabs below. Codes are here
